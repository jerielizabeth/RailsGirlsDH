We're bringing RailsGirls to the Digital Humanities

We all know there are too few women creating the code that runs digital humanities projects. And for many of us, the largest problem is knowing where to start and what questions to ask.

RailsGirls is an opportunity to learn about the technology behind web applications while setting up your own Ruby on Rails application.

We will be hosting a RailsGirls event here at George Mason on September 7, with a preliminary installation "party" on September 6.

If you are interested in attending, volunteering, or supporting this event, we want to hear from you
